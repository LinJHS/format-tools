# 格式匠

一款面向学术与专业写作人员的 Markdown 转 Word 工具，提供零配置、专业级的文档转换体验。

## 📋 功能特性

### 核心功能
- **Markdown 转 Word**：支持 `.md`、`.zip` 等多种格式，支持文本框直接输入，一键转换为 `.docx`。
- **丰富模板**：内置多款精美模板。
- **AI 格式修复**：自动修复 Markdown 格式问题，提升输出质量。

### 平台特性

- **跨平台支持**：Windows、macOS、Linux 完整支持
- **离线转换**：核心转换功能无需网络连接

## 🚀 快速开始

### 前置要求
- Node.js 16+
- Rust 1.70+
- pnpm

### 安装与运行

```bash
# 安装依赖
pnpm install

# 开发模式运行
pnpm tauri dev

# 构建发布版本
pnpm tauri build
```
